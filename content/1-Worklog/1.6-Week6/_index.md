# Week 6: Automatic Workstation Lock and Unlock Logic

**Date:** 02/06/2026 - 07/06/2026

During the sixth week, I focused on designing the operating logic for the automatic workstation lock and unlock process.

## Tasks Completed

- Proposed the idea for the automatic workstation lock and unlock workflow.
- Designed the closed-loop operation logic for workstation control.
- Planned the process for locking a workstation when a member runs out of usage time.
- Planned the process for locking a workstation when a member logs out.
- Analyzed how the workstation status should be updated in the system.

## Result

After completing this week, the basic logic for automatic workstation control was defined.

This logic helped prepare the next step of building the lock screen interface and connecting it with the CyberNet system workflow.
